The Web

HTML is Hyper Markup Language. It is a document that contains all sort of content
but is the basis for everything on the web.

Your browser makes request that servers reply to with content. HTTP is the protocol
that the browser uses to speak to servers.

HTML is made up of text content like what I am writing right now. It is also made up
of links to content and markup which is the way a site looks.

HTML

HTML markup is made up of tags. Which are made up of < Name > Contents Then
closing with < /name >

The B tag makes anything inside of the tag BOLD

The EM tag means emphasis aka ITALICS.

HTML Attributes

Anchor tag is a link tag that is GREAT for checking the NEWS

The Image tag is how you would link image content to the page!! Image tags are VOID tags so they
do not need a closing tag.



Whitespace - The br tag allows for a line break. It is also a void tag that does not need a closing tag.

In comparsion a P tag (paragrah) is not a void tag and requires a closing tag.

The br tag in considered an inline tage which just breaks the line in comparsion to
the P tag which is a block tag that has an invisible box.

Inline elements include B em br and even img tags

Span is an inline element that contains it content. It allows you to add style to the content through CSS.

Div is a block element that contains its content. The div element will create a separate line.

HTML Structure

An HTML document includes a head (meta and java etc.) with a title which would
be shown in the browser tab. Then it has the body content which would contain
content like the kind I am making right now.
